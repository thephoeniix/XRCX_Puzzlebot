# XRCX_Puzzlebot

> Stack de navegación autónoma en ROS 2 para el robot Puzzlebot, adaptado desde TurtleBot3 con soporte para SLAM, simulación en Gazebo y control de navegación propio.

## Objetivo

Este proyecto tiene como fin desarrollar un stack de navegación autónoma en **ROS 2** para el robot **Puzzlebot**, migrando desde TurtleBot3 hacia una implementación propia. Los objetivos específicos son:

- Crear la estructura inicial del proyecto de navegación en ROS 2 para Puzzlebot.
- Integrar el modelo URDF/Xacro del robot dentro de un paquete propio.
- Integrar el mundo y el entorno de simulación generados en la actividad anterior.
- Adaptar la parte de SLAM del stack para el robot y el mapa propio.
- Preparar la base del modo de navegación.
- Organizar el proyecto siguiendo buenas prácticas de paquetes en ROS 2.
- Iniciar el control de versiones mediante un repositorio público en GitHub.
